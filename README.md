# SMOODIE
Smoodie는 사용자의 기분과 감정을 간편하게 기록하고 분석할 수 있는 무드트래커 앱입니다. 
실시간 데이터 업데이트와 직관적인 시각화를 통해 기분 변화를 손쉽게 추적할 수 있습니다.
